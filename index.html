GPX track visualizer

Basically i got frustrated with searching for a GPX visualizer that can generate Strava like heat maps from several .gpx files. This is a project written in C++ using SDL2 for GPU accelerated rendering. It aims to add click, drag and zoom movement similar to osm.net. Source code: github
