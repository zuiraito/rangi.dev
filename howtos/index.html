Compress a Directory

how to back up: tar --use-compress-program="zstd -T0 -6" -cf /path/to/backup/location/backup-$(date -Iminute).tar.zst /path/to/data/that/you/want/to/back/up/
split the file for easier upload split -b 5G -d --verbose backup.tar.zst backup.tar.zst.part
combine files again cat backup.tar.zst.part* > backup.tar.zst
uncompress tar -I zstd -xf backup.tar.zst

pdftk * cat output output.pdf

Stuff to do with Networks

Set up a WIFI Connection in the Terminal

List available networks nmcli d wifi l
Connect to network nmcli d wifi c [ssid] p [password]
Switch network nmcli c up [ssid]
Delete network nmcli c delete [ssid]

Set up Eduroam in the Terminal

nmcli con add type wifi con-name "eduroam" ifname wlan0 ssid "eduroam" \
802-1x.eap peap \
802-1x.phase2-auth mschapv2 \
802-1x.identity "my_username" \
802-1x.password "my_password" \
wifi-sec.key-mgmt wpa-eap \
802-1x.ca-cert "/etc/ssl/certs/ca-certificates.crt"

Then you can use nmcli to start the connection: nmcli con up "eduroam"
