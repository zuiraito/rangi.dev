Create a bootable Installation Media

Storage device finden

lsblk

Wenn .img.xz muss man die Datei decompressen

xy -d path/to/image.img.xz

Flashen

sudo dd if=path/to/your/image.img of=/dev/sdX bs=4M status=progress && sync

Map Caps-Lock to Escape

In my opinion Caps-Lock has no right being that big of a key in a that well readable spot. The key would be better suited for the escape or control function.

Changing the key mapping depends heavily on the display manager. Sometimes it is possible to change the key mapping in the BIOS configuration.

Gnome

Gnome gsettings set org.gnome.desktop.input-sources xkb-options "['caps:escape']"
Hyprland add the line kb_options = caps:swapescape to the input {} section of your hyprland.conf .
MacOS In MacOS the mapping can be changes in the normal settings but it has to be changes for every single (physical) keyboard you use.

Remove the Firefox tab bar entirely

  1. Enable user Chrome: enter about:config into the address bar, accept the risks.
  2. Search for toolkit.legacyUserProfileCustomizations.stylesheets and toggle it to true.
  3. Locate the profile folder by clicking on the "☰ " then "Help" then "Troubleshootin│g Information" and open it.
  4. Create a new directory named chrome.
  5. Inside the chrome directory, create two new files, userChrome.css and userContent.css (case sensitive).
  6. Add the line #TabsToolbar {visibility: collapse;} or TabsToolbar { visibility: collapse !important; } if you only want to hide the tab bar and leave the navigation bar.
  7. Save the file and restart Firefox.
