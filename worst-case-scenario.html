No special hardware, no special software

The simplest solution for secure (in this case secure means no one expect of the sender and receiver can read the conversation) communication is to manually encrypt messages with PGP. In this way any messaging app/protocol (Signal, Whatsapp, E-Mail) can be used to carry the encrypted message by exchanging public PGP keys. With this approach all responsibility and control is in the hands of the users without needing to trust any provider claiming well implemented end to end encryption
